# 💰 Expense Tracker

A simple and interactive Expense Tracker web application built with **Python, Streamlit, SQLite, Pandas, and Matplotlib**. The application helps users manage their income and expenses, track balances, and visualize financial data through reports and charts.

## 🚀 Features

- ➕ Add new income and expense transactions
- ✏️ Edit existing transactions
- 🗑️ Delete transactions
- 📋 View complete transaction history
- 📊 Generate financial reports and charts
- 💵 Track total income, expenses, and current balance
- 💾 Store data using SQLite database

## 🛠️ Tech Stack

- **Python**
- **Streamlit**
- **SQLite**
- **Pandas**
- **Matplotlib**

## 📈 Functionalities

- Dashboard with total income, expenses, and balance.
- Transaction history table.
- Income and expense management.
- Interactive financial reports using charts.
